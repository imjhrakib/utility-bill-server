Utility Bill Payment Server

This is a Utility Bill Payment Server built with Express.js, MongoDB, and CORS. Customers can pay and manage their utility bills like Water, Gas, Electricity, and Internet.

Live Server: https://utility-bill-server.vercel.app/

---

Features

- View all available utility bills.
- Customers can pay bills and manage their own bills.
- CRUD operations for customer bills (Create, Read, Update, Delete).
- Filter bills by user email.
- Backend ready for PDF/CSV report generation.

---

Technologies Used

- Node.js – JavaScript runtime
- Express.js – Backend framework
- MongoDB – Database
- CORS – Enable cross-origin requests
- dotenv – Environment variables
